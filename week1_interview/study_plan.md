## 💡 面试技巧

### 回答框架

**概念题**：定义 + 特点 + 例子
```
Q: 什么是协程？
A: 协程是用户态轻量级线程（定义），
   特点是协作式调度、单线程并发（特点），
   比如 Python 的 async/await（例子）
```

**代码题**：思路 + 实现 + 边界
```
Q: 如何限制并发？
A: 我的思路是用 Semaphore（思路），
   代码是这样写的...（实现），
   需要注意信号量要在循环外创建（边界）
```

### 遇到不会的题

1. 不要慌，先复述问题确认理解
2. 说已知的部分，诚实说不知道的部分
3. 尝试类比或迁移已知知识

```
面试官：I/O 多路复用的原理是什么？
你：这个我不是很熟悉，但我理解它应该是
   让单个线程能监控多个 I/O 连接的技术，
   类似 epoll 这样的机制...
```
