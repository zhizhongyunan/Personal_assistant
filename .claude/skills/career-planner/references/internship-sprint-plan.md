# 实习冲刺计划模板

## 适用场景
- 目标：3-6个月内找到暑期实习/日常实习
- 人群：在校学生（研一/研二为主）

## 计划周期
12周（约3个月）

---

## 时间线总览

| 阶段 | 周次 | 核心任务 |
|------|------|----------|
| 基础期 | 第1-4周 | 技能巩固 + 项目启动 |
| 强化期 | 第5-8周 | 项目收尾 + 刷题强化 + 首批投递 |
| 冲刺期 | 第9-12周 | 面试准备 + 面试冲刺 |
| 入职期 | 第13周+ | 实习入职 |

---

## 第一阶段：基础期（第1-4周）

### 目标
- 巩固核心技术基础
- 确定项目方向并启动
- 养成刷题习惯

### 每周任务

#### 第1周
- [ ] 确定项目方向（NLP/推荐/CV/大模型选择一个）
- [ ] 调研数据集（Kaggle/UCI/天池）
- [ ] LeetCode 数组题型 15 题
- [ ] 回顾 PyTorch 基础

#### 第2周
- [ ] 项目代码框架搭建
- [ ] 数据预处理完成
- [ ] LeetCode 字符串题型 15 题
- [ ] 整理机器学习笔记

#### 第3周
- [ ] 项目模型训练
- [ ] 基础评估指标实现
- [ ] LeetCode 链表/树题型 15 题
- [ ] 学习 Transformer 基础

#### 第4周
- [ ] 项目初步完成
- [ ] LeetCode 栈/队列 15 题（累计60题）
- [ ] 开始投递前准备（注册平台、完善GitHub）

### 产出要求
- LeetCode：60 题
- 项目进度：50%
- 平台注册：完成

---

## 第二阶段：强化期（第5-8周）

### 目标
- 项目完全上线
- 简历完成并开始投递
- 算法题达到阈值

### 每周任务

#### 第5周
- [ ] 项目部署到 GitHub
- [ ] README 完善（问题定义、方法、结果）
- [ ] LeetCode 栈/队列/堆 20 题（累计80题）
- [ ] 开始写技术博客

#### 第6周
- [ ] 简历第一版完成
- [ ] LeetCode 动态规划入门 20 题（累计100题）
- [ ] 背八股文（机器学习）
- [ ] 项目补充：尝试改进/优化

#### 第7周
- [ ] 简历优化完成
- [ ] 开始投递（10+ 公司）
- [ ] LeetCode 动态规划进阶 20 题（累计120题）
- [ ] 准备项目deep dive

#### 第8周
- [ ] 投递 20+ 公司
- [ ] LeetCode 复习高频题型（累计130题）
- [ ] 开始模拟面试
- [ ] 项目增加部署/Demo（如Gradio/Streamlit）

### 产出要求
- 项目：1-2个完整项目上线
- LeetCode：130 题
- 简历：完成并优化
- 投递：20+ 公司

---

## 第三阶段：冲刺期（第9-12周）

### 目标
- 面试密集准备
- 持续投递
- 查漏补缺

### 每周任务

#### 第9周
- [ ] 持续投递（每周10+）
- [ ] 机器学习八股文复习
- [ ] 深度学习八股文复习
- [ ] 项目细节完全熟悉

#### 第10周
- [ ] 面试复盘（每次面试后记录问题）
- [ ] 八股文查漏补缺
- [ ] LeetCode 保持手感（每周20题）
- [ ] 系统设计基础

#### 第11周
- [ ] 模拟面试强化
- [ ] 项目展示优化
- [ ] 行为面准备（宝洁八大问）
- [ ] 持续投递

#### 第12周
- [ ] 查漏补缺
- [ ] 面试邀请少的调整策略
- [ ] 准备入职材料（如已有Offer）

### 产出要求
- 投递：50+ 公司
- 面试：5-15 次
- Offer：1-3 个

---

## 关键时间节点

| 时间 | 动作 | 优先级 |
|------|------|--------|
| 第1周周日 | 确定项目方向 | 必做 |
| 第4周周日 | 项目50% + 60题 | 必做 |
| 第5周周一 | 注册Boss/实习僧/牛客 | 必做 |
| 第6周周日 | 简历完成 | 必做 |
| 第7周周一 | 开始投递（宜早不宜晚） | 必做 |
| 第8周周日 | 130题 + 20公司投递 | 必做 |
| 第10周 | 第一次面试 | 预期 |
| 第12周 | 拿到Offer | 目标 |

## 每日时间分配（参考）

### 在校学生（每天4-6小时）

| 时间段 | 内容 |
|--------|------|
| 早上/午间 0.5h | 八股文/论文 |
| 下午/晚上 2h | 项目 |
| 晚上 1.5h | LeetCode |
| 碎片0.5h | 技术博客/行业动态 |

### 全天脱产（每天8-10小时）

| 时间段 | 内容 |
|--------|------|
| 上午 2h | LeetCode |
| 下午 3h | 项目/八股文 |
| 晚上 2h | 投递/复盘 |
| 碎片1h | 博客/动态 |

---

## 风险与应对

| 风险 | 预防/应对 |
|------|------------|
| 项目做不完 | 第4周后优先完成项目，刷题可放缓 |
| 投递无回应 | 检查简历问题，海投50+后再调整 |
| 面试少 | 扩大范围，投递中小厂 |
| 算法题不够 | 重点刷高频题型200题 |
| 时间不够 | 减少社交，固定学习时间 |

---

## 面试高频问题分类

### 算法题（必须掌握）
- 数组/字符串
- 动态规划
- 树/二叉树
- 链表
- 栈/队列

### 八股文
- 机器学习：过拟合、正则化、梯度下降、交叉验证
- 深度学习：反向传播、激活函数、BN、Dropout
- Transformer：QKV、Attention、位置编码
- 项目相关

### 项目问题
- 项目背景与动机
- 技术选型
- 遇到的最大挑战
- 如何解决/优化

## 实习转正路径

1. **入职第1月**：熟悉业务，快速产出
2. **入职第2月**：独立负责功能，争取转正答辩材料
3. **入职第3月**：转正答辩，拿到_return Offer
