## 📅 4 个月详细学习计划（2026.03 - 2026.07）

### 第一阶段：基础巩固（第 1-2 周）

**目标**：夯实 Python 基础和 LangChain 核心概念

| 周次 | 学习内容 | 产出物 | 检查点 |
|------|----------|--------|--------|
| 第 1 周 | - Python 异步编程（async/await）<br>- LangChain 核心组件复习<br>- ChatModel 深入理解 | 完成 3 个异步调用 Demo | 能独立编写异步 LLM 调用 |
| 第 2 周 | - Prompt Template 高级用法<br>- Output Parser 详解<br>- 基础 Memory 实现 | 实现带记忆的对话机器人 | 理解 ConversationBufferMemory |

**推荐资源**：
- LangChain 官方文档：https://python.langchain.com/
- B 站：LangChain 中文教程

---

### 第二阶段：RAG 核心技术（第 3-6 周）⭐⭐⭐⭐⭐

**目标**：掌握 RAG 全流程，能独立搭建知识库问答系统

| 周次 | 学习内容 | 产出物 | 检查点 |
|------|----------|--------|--------|
| 第 3 周 | - 向量数据库基础（Chroma）<br>- 文本分块策略<br>- Embedding 原理 | 搭建本地 Chroma 知识库 | 能完成文档向量化存储 |
| 第 4 周 | - Retriever 配置与优化<br>- 向量检索 + 关键词检索<br>- RAG 基本流程 | 实现文档问答 Demo | 能准确检索相关文档 |
| 第 5 周 | - 高级 RAG 技巧（多路召回、重排序）<br>- Rerank 模型使用 | 优化检索准确率 | 检索准确率>80% |
| 第 6 周 | - RAG 项目实战：个人知识库问答 | 完整 RAG 项目 | 可作为面试项目展示 |

**推荐项目**：
- 个人笔记知识库问答
- PDF 文档智能问答

---

### 第三阶段：Agent 进阶（第 7-10 周）⭐⭐⭐⭐⭐

**目标**：深入理解 Agent 架构，掌握复杂 Agent 开发

| 第 8 周 | - LangGraph 状态图<br>- 多节点 Agent 设计<br>- 条件路由 | 实现多步骤工作流 | 理解 StateGraph 核心概念 |
| 第 9 周 | - Agent 规划能力（ReAct/Plan-and-Execute）<br>- Reflection 机制 | 实现规划型 Agent | 能完成多步骤任务分解 |
| 第 10 周 | - Agent 项目实战：智能助手 | 完整 Agent 项目 | 可作为面试项目展示 |

**推荐项目**：
